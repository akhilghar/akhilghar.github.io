ASE 370C: Feedback Control Systems - Spring 2026

Real-time Feedback Control for autonomous aircraft pursuit and interception systems.

Overview

The aim of this project is to develop a feedback control system for an autonomous aircraft pursuit and interception scenario. The project focuses on real-time implementation of control algorithms to achieve precise and efficient target tracking and interception.

Block Diagram of Control Architecture!

This diagram illustrates the flow of signals and information throughout the control system. We chose a nested loop configuration to allow for more precise control of the aircraft's dynamics. The outer loop is responsible for tracking the standoff distance between the target and ownship as well as deriving the reference state trajectories every 0.5 seconds, while the inner loop focuses on tracking the aircraft's attitude and ensuring smooth flight every 5 ms. This architecture allows for efficient handling of disturbances and uncertainties in the environment, ultimately improving the performance of the pursuit and interception system.
