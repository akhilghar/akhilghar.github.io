Real-time Trajectory Optimization for Autonomous Ground Robots.

Overview

Designed and integrated a lightweight (<1kg) quadcopter platform capable of autonomously following a target object in real time.

Control Architecture

Implemented a Linear Quadratic Regulator (LQR) in MATLAB to stabilize and track desired trajectories.

Vision System

Used OpenCV on Raspberry Pi for low-latency (<5ms) image processing and target tracking.

System Integration

Designed custom 14" x 14" quadcopter chassis integrating flight controller, ESCs, battery, and onboard compute.
