Tracker Drone Project

Full-system UAV integrating modeling, LQR control, state estimation, aerodynamic simulation, and hardware implementation.

System Overview

The Tracker Drone is a fully integrated quadcopter platform designed for autonomous object tracking. The project combines control theory, state estimation, aerodynamic modeling, and embedded systems into a cohesive UAV architecture.

Control Architecture

Full-state feedback LQR controller with Kalman-based state estimation and complementary IMU filtering.

A continuous-time state-space model of the quadcopter dynamics was linearized and controlled using a Linear Quadratic Regulator (LQR). The controller computes optimal state feedback gains minimizing quadratic state and control effort cost.

Results

The integrated LQR + Kalman architecture demonstrated stable tracking and rapid convergence with minimal steady-state error. Simulation results validated control effort constraints and real-time feasibility for embedded deployment.
